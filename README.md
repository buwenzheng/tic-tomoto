# 番茄时钟 (Tic-Tomoto)

![Version](https://img.shields.io/badge/version-0.1.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

一个功能丰富的番茄工作法计时器应用。

## 功能特点

- 🍅 专业的番茄钟计时功能
- 📋 任务管理与追踪
- 🌙 深色模式支持
- 📊 工作统计与分析
- ⌨️ 全局快捷键支持
- 🔔 自定义提醒设置

## 开发

```bash
# 克隆项目
git clone https://github.com/yourusername/tic-tomoto.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
